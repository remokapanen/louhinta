Chapter 2 Yomitan

Yomitan on browser-lisäosa, joka mahdollistaa nopeasti sanojen merkityksen tarkistamisen ja niiden siirtämisen Ankiin.

2.1 Asennus

Lataa Yomitan käyttämällesi browserille (Chrome ja Firefox sopivimmat) https://yomitan.wiki/

2.2 Asetukset

Lataa seuraava tiedosto: Download Yomitan Settings JSON

  • Backup -> Import settings
  • Valitse Active profile -> japani, jos opiskelet japania; muussa tapauksessa valitse Muut

2.3 Valitse opiskelemasi kieli

General -> Language

2.4 Yomitanin sanakirjat

Jos opiskelet japania, lataa seuraava tiedosto (vaihtoehtoisesti voi myös ladata suoraan Yomitanin ehdottamat sanakirjat kohdasta get recommended dictionarys)

  • Backup -> Import dictionary collection

  • Aseta Configure installed and enabled dictionarys -kohdasta tämän jälkeen kaikki yllä olevat sanakirjat päälle

Jos opiskelet jotain muuta, Yomitan suosittelee sopivat sanakirjat.

  • Get recommended dictionarys -> lataa suositellut.

  • Tämän jälkeen Configure installed and enabled dictionarys -> laita kyseinen sanakirja päälle.

2.5 Ankiin yhdistäminen ja korttien säätäminen

  • Enable Anki integration (Ankin tulee olla päällä samaan aikaan)

  • Configure Anki flashcards -> Deck-kohtaan louhintaa varten luomasi pakan nimi

  • Maindefinition-kohtaan tärkein lataamasi sanakirja (Japanin opiskelijoille jo valmiiksi asetettu, muille kielille laita tähän kohtaan lataamasti sanakirjan nimi)
